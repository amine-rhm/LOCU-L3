## Description

**LOCU-L3** est une application web full-stack développée pour le projet de fin d'études en Licence 3 Informatique.  
Elle permet de gérer les biens immobiliers, les utilisateurs et les réservations de manière efficace et intuitive.

---

## Fonctionnalités principales

- Gestion des utilisateurs (inscription, connexion, rôles)  
- Gestion des biens immobiliers (ajout, modification, suppression)  
- Gestion des réservations et suivi des disponibilités  
- Interface responsive avec Tailwind CSS  
- Backend robuste avec Node.js et Express  
- Frontend moderne avec React et TypeScript
